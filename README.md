## Develop

[![Gitpod ready-to-code](https://img.shields.io/badge/Gitpod-ready--to--code-blue?logo=gitpod)](https://gitpod.io/#https://github.com/OVaaS/ovaas-front)

This project use pnpm as package manager.

### Install pnpm
```
npm i -g pnpm
```

### Start dev server
```
pnpm dev
```

### Build for production
```
pnpm build
```
